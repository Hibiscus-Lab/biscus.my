Jom Belajar Hack (Intermediate)

Overview

Penetration testing, also known as Pen Test, is the most common technique for testing the security of web applications. Web Application Penetration Testing involves simulating internal or external unauthorised attacks to gain access to sensitive data. With a weekly intensive online workshop, we will go from I don't even know how to install it to actually understanding web application pentest. You will study and practise Web Pentest across multiple modules. Using Portswigger and bWapp labs, we will conduct live pentests and provide step-by-step instruction.
